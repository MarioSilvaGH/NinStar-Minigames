A collection of extensions, utilities and games for GameMaker.

```bash
git clone https://github.com/ninstar/yyProjects.git
```


|IDE				|Version			|															|
|-------------------|-------------------|-----------------------------------------------------------|
|GameMaker: Studio	|1.4.9999			|For ``.gmx`` and ``.gmez`` files.	|
|GameMaker Studio 2	|2.2 or newer 	|For ``.yyp`` and ``.yymp`` files.	|

> You can convert a .gmx project into a .yyp as well as convert a .gmez extension into a .yymp one, but it would still require manual work to fix any potential incompatibility issue.

## Extensions

+ **[audio_loop_sound][loop]** - Play multiple audios using sampled loop points.

## Utilities

+ **[Project Cleaner][cleaner]** - Scan and remove unused assets and graphics from projects made with GameMaker Studio 2.

## Games

+ **[Nin★Shooter][shooter]** - Side-scrolling shooter with a power-up system.
+ **[Nin★Platformer][platformer]** - A Super Mario Bros. styled platformer.
